## Tổng quan
Đây là một topdown shooter game
## Cách chơi
Mục tiêu trò chơi là di chuyển nhân vật và cố gắng tiêu diệt nhiều kẻ địch nhất có thể
# Các phím bấm
|   Key   | Action                  |
|:-------:| ----------------------- |
|   `A`   | Move left               |
|   `D`   | Move right              |
|   `W`   | Move up                 |
|   `S`   | Move down               |
|   `E`   | Shoot                   |
